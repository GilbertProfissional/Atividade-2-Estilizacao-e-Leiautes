# Atividade-1-Desenvolvimento-Front-end
Este projeto consiste na criação de um site institucional para uma ONG Digital, desenvolvido como parte da Atividade 1 de Desenvolvimento Front-end. O objetivo é aplicar conhecimentos de HTML, CSS e boas práticas de acessibilidade e semântica web.

Sobre o projeto
O site apresenta cinco páginas principais:
Página Inicial: apresenta a ONG, sua missão, visão, valores, galeria de imagens, vídeos e áudios institucionais.
Sobre: detalha informações sobre a ONG, seu histórico e objetivos.
Projetos: lista os projetos sociais em andamento, com oportunidades de voluntariado e informações sobre doações.
Cadastro: formulário para voluntários e doadores, coletando informações pessoais e endereço.
Contato: formulário para envio de mensagens à ONG, possibilitando comunicação direta com os visitantes.
O projeto foi desenvolvido respeitando a semântica HTML5, usando tags como header, main, section, aside e footer para melhorar a estrutura e acessibilidade do site. Também foram aplicadas práticas de validação de formulários e links internos para facilitar a navegação entre as páginas.

Tecnologias utilizadas
HTML5
CSS3
Navegação responsiva para desktop e mobile
Validações
Todas as páginas foram validadas no W3C Validator, garantindo que o código está correto e acessível. As imagens e links possuem textos alternativos e atributos apropriados para melhorar a experiência do usuário.

Estrutura do projeto
O projeto está organizado da seguinte forma:
index.html – Página inicial
sobre.html – Página sobre a ONG
projetos.html – Página de projetos sociais
cadastro.html – Formulário de cadastro
contato.html – Formulário de contato
styles.css – Arquivo de estilos
imagens/ – Pasta com imagens utilizadas no site
videos/ – Pasta com vídeos institucionais
audios/ – Pasta com áudios institucionais
Validacoes W3C.docx – Arquivo com prints de validação das páginas

Objetivo da atividade
A atividade teve como objetivo desenvolver um site funcional, acessível e validado, aplicando conceitos de estrutura semântica, navegação, formulários e boas práticas de front-end.
