# Atividade-2-Estilização e Leiautes
Descrição

Este projeto faz parte das atividades da disciplina Desenvolvimento Front-end para Web, com o objetivo de aplicar conceitos de CSS3 para estilização e construção de leiautes responsivos.
O trabalho envolve a estruturação semântica do HTML e a criação de um design moderno e funcional, adaptável a diferentes tamanhos de tela (desktop e mobile).
O projeto está dividido em diretórios para melhor organização dos arquivos:
reset.css → Remove estilos padrão do navegador.
variables.css → Define variáveis globais (cores, fontes, espaçamentos).
layout.css → Controla o posicionamento dos elementos principais.
navigation.css → Cuida da barra de navegação e menus.
components.css → Estiliza componentes reutilizáveis.
utilities.css → Classes utilitárias para ajustes rápidos.
Tecnologias Utilizadas:
HTML5 (estrutura semântica)
CSS3 (estilização e responsividade)
Visual Studio Code com Live Server
Git e GitHub para versionamento e hospedagem do código
